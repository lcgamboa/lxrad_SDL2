LXRAD Layout HELP

The menu commands are:
File
    New Session       Open project or create new project
    New Window     Add window to project
    Save                      Save project
    Exit                       Exit layout

Edit
    Options                Compiler Options
    Clear                     Clear window
    Delete                   Delete selected object

Project
    Run                        Run project
    Build                      Build project
    Edit Source           Edit Source code of current window
    Debug                    Run Debug
 
Windows                   Select window for edition

Help
    Contents                Help contents

The key commands are:

del                                             delete selected object
shift+(up,down,left,rigth)   resize the selected object +/- 1 pixel
crtl+(up,down,left,rigth)     move the selected object +/- 1 pixel
